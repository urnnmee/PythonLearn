Строки и списки в Python

Строки

Строки в Python — это последовательности символов, которые записываются в одинарных (' ') или двойных (" ") кавычках. Они неизменяемы, то есть после создания строки нельзя изменить отдельные её символы, но можно создавать новые строки на основе старых.

  • Индексация: каждый символ в строке имеет свой индекс, начиная с 0. Например, в строке "Python" символ P — это s[0], а n — s[5]. Отрицательные индексы считаются с конца: s[-1] — последний символ.
  • Срезы: чтобы взять часть строки, используется синтаксис [начало:конец:шаг]. Например, "Python"[1:4] вернёт "yth". Если шаг не указан, он равен 1.
  • Методы строк: Python предлагает множество встроенных методов для работы со строками:
    • .upper() — переводит строку в верхний регистр.
    • .lower() — в нижний регистр.
    • .replace("старое", "новое") — заменяет подстроку.
    • .strip() — убирает пробелы в начале и конце строки.
    • .split() — разбивает строку на список по указанному разделителю (например, пробелу).

Пример:


s = "Hello, World!"
print(s[0])         # H
print(s[7:12])      # World
print(s.upper())    # HELLO, WORLD!
print(s.split())    # ['Hello,', 'World!']

Списки

Списки — это упорядоченные изменяемые коллекции данных, которые записываются в квадратных скобках [ ]. В список можно помещать элементы разных типов: числа, строки, даже другие списки.

  • Индексация и срезы: работают так же, как со строками. Например, в списке [10, 20, 30] элемент 20 доступен по индексу [1], а срез [0:2] вернёт [10, 20].
  • Изменение списков: в отличие от строк, списки можно менять. Например, lst[0] = 5 заменит первый элемент.
  • Методы списков:
    • .append(x) — добавляет элемент x в конец списка.
    • .insert(i, x) — вставляет элемент x на позицию i.
    • .remove(x) — удаляет первое вхождение элемента x.
    • .pop(i) — удаляет и возвращает элемент по индексу i (по умолчанию последний).
    • .sort() — сортирует список по возрастанию.

Пример:


numbers = [1, 2, 3]
numbers.append(4)       # [1, 2, 3, 4]
numbers.insert(1, 10)   # [1, 10, 2, 3, 4]
numbers.remove(2)       # [1, 10, 3, 4]
print(numbers.pop())    # 4, список станет [1, 10, 3]
print(numbers[1:3])     # [10, 3]

Полезные особенности

  • Строки можно соединять (конкатенировать) с помощью +: "Hello" + " World" даст "Hello World".
  • Списки тоже можно объединять: [1, 2] + [3, 4] станет [1, 2, 3, 4].
  • Для проверки наличия элемента используется оператор in: "a" in "cat" вернёт True, а 5 in [1, 2, 3] — False.

Итог

Строки и списки — мощные инструменты для работы с данными. Строки хороши для обработки текста, а списки — для хранения и изменения наборов данных. Оба типа поддерживают индексацию и срезы, но списки более гибкие благодаря своей изменяемости.

Условные операторы и циклы в Python

Введение

Условные операторы и циклы являются основными элементами для написания логики в программах на Python. Понимание их работы позволяет создавать более сложные и интерактивные приложения.

1. Условные операторы

Условные операторы позволяют программе принимать решения и выполнять различные действия в зависимости от заданных условий. В Python используются операторы if, elif и else.

Синтаксис:


x = 10
if x > 0:
print("Число положительное")
elif x == 0:
    print("Число равно нулю")
else:
    print("Число отрицательное")

2. Циклы

Циклы позволяют выполнять один и тот же код несколько раз. В Python есть два основных типа: for и while.

Цикл for

Цикл for используется, когда известно, сколько раз нужно повторить код.


fruits = ["яблоко", "банан", "груша"]
for fruit in fruits:
    print(fruit)

Цикл while

Цикл while продолжает выполняться, пока заданное условие истинно.


count = 0
while count < 5:
    print(count)
    count += 1
