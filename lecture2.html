Переменные, типы данных и операторы

Введение

В программировании переменные, типы данных и операторы — это основополагающие концепции, которые позволяют нам хранить, обрабатывать и манипулировать данными. Понимание этих понятий является ключом к успешному программированию.

1. Переменные

Что такое переменная? Переменная — это именованное хранилище для данных. Она позволяет сохранять значения, которые могут изменяться в процессе выполнения программы.

Объявление переменной:

В большинстве языков программирования переменные объявляются с помощью ключевых слов, таких как var, let, const (в JavaScript), int, float, string (в C/C++), и т.д.

let age = 25; // Объявление переменной age и присвоение значения 25
const name = "Alice"; // Объявление константы name

2. Типы данных

Типы данных определяют, какой вид значений может хранить переменная и какие операции можно выполнять с этими значениями. Рассмотрим основные типы данных:

2.1 Числа

  • Целые числа (integer): Например, -1, 0, 42.
  • Вещественные числа (float): Например, 3.14, -0.001.
let integerNumber = 10; // Целое число
let floatNumber = 3.14; // Вещественное число

2.2 Строки

Строки представляют собой последовательности символов и используются для хранения текстовой информации.

let greeting = "Hello, World!"; // Строка

2.3 Булевы значения

Булевы значения могут принимать только два состояния: true (истина) или false (ложь). Они часто используются для управления логикой программы.

let isActive = true; // Булевое значение

3. Операторы

Операторы — это специальные символы, которые выполняют операции над переменными и значениями.

3.1 Арифметические операторы

Арифметические операторы используются для выполнения математических операций.

  • + — сложение
  • - — вычитание
  • * — умножение
  • / — деление
  • % — остаток от деления
let sum = 5 + 10; // 15
let product = 5 * 10; // 50

3.2 Логические операторы

Логические операторы используются для выполнения логических операций.

  • && — логическое И
  • || — логическое ИЛИ
  • ! — логическое НЕ
let a = true;
let b = false;
let result = a && b; // false (поскольку оба условия не истинны)
