Условные операторы и циклы в Python

Введение

Условные операторы и циклы являются основными элементами для написания логики в программах на Python. Понимание их работы позволяет создавать более сложные и интерактивные приложения.

1. Условные операторы

Условные операторы позволяют программе принимать решения и выполнять различные действия в зависимости от заданных условий. В Python используются операторы if, elif и else.

Синтаксис:


x = 10
if x > 0:
    print("Число положительное")
elif x == 0:
    print("Число равно нулю")
else:
    print("Число отрицательное")

2. Циклы

Циклы позволяют выполнять один и тот же код несколько раз. В Python есть два основных типа: for и while.

Цикл for

Цикл for используется, когда известно, сколько раз нужно повторить код.


fruits = ["яблоко", "банан", "груша"]
for fruit in fruits:
    print(fruit)

Цикл while

Цикл while продолжает выполняться, пока заданное условие истинно.


count = 0
while count < 5:
    print(count)
    count += 1

Итог

С помощью условных операторов, таких как if, elif и else, мы можем управлять потоком программы. Циклы for и while позволяют создавать повторяющиеся действия, что является основой для написания более сложных и полезных программ.
